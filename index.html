Offline SpriteKit game

Grass Factory

A cheerful mowing-and-production game where automatic mowers clear sunny fields, trucks haul the harvest, and factory machines turn grass into upgrades.

  • 50 levels
  • Offline play
  • iOS 13+
  • Landscape

Mow, haul, process, upgrade.

Every stage is a compact production puzzle. Keep the mowing line moving, make room in the trucks, feed the factory, and spend coins on upgrades that help you finish with a stronger star rating.

Automatic mowing Mowers cut reachable grass in satisfying row segments while routing around stones and blocked tiles.
Factory timing Trucks shuttle harvested grass into processing machines, and each completed batch pays coins.
Level-local upgrades Improve mower speed, storage, trucks, and processing power for the current challenge.

Built for iOS

Grass Factory is built with SpriteKit and Objective-C. It supports iOS 13 and later, runs in landscape orientation, and is designed for every iPhone model that supports iOS 13.

Privacy first

The game is fully offline, stores progress locally on the device, and does not require an account. See the privacy policy for details.
